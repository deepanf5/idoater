# idolater application

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.5.

## About this application

idolater is a simple and modern task management application that helps users organize and track their daily tasks. Users can create, update, and manage tasks while monitoring their progress throughout the day.

Built with Angular Signals and Signal Forms for reactive state management and form handling, and powered by Supabase for authentication, database storage, and real-time synchronization.

Expense Tracking & Categorization — Easily record, organize, and categorize expenses to monitor spending patterns, manage budgets, and gain better financial insights.


## Features
Add, edit, and delete daily tasks
Track task completion status
Expense tracker
User authentication and secure data storage
Real-time updates with Supabase
